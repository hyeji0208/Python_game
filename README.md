# 대학생 러닝 게임

![게임 스크린샷](graphic/main.png)

> Ursina 엔진을 사용하여 만든 2D 러닝 게임입니다. 장애물과 아이템을 피해 점수를 올리는 게임입니다.

---

## 🎮 게임 설명

- **목표**: 점수를 높이며 끝까지 살아남기!
- **조작**:
  - `← / →` 또는 `A / D`: 좌우 이동
  - `Space`: 점프
- **게임 요소**:
  - 장애물: 과제, 노트북, 팀플 팀원 등
  - 아이템:
    - ❤️ 체력 회복
    - 📄 휴학 신청서 (무적 상태)

---
